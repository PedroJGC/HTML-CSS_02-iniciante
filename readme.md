# 🚀 Curso Fullstack HTML e CSS - Rocketseat

Bem-vindo ao **Curso Fullstack HTML e CSS** da Rocketseat! Neste curso, você aprenderá os fundamentos essenciais para desenvolver páginas web modernas e responsivas.

---

## 📚 Conteúdo do Curso

### 🏗️ Iniciando com HTML e CSS
Neste curso, você aprenderá a estruturação e estilização de páginas web utilizando **HTML** e **CSS** desde o nível iniciante.

---

## 📌 Módulos do Curso

### 📜 **Fundamentos HTML**
📦 **Módulo**
- 🎥 **32 aulas**
- 📝 **5 Atividades**
- ⏳ **Duração: 1h 28min**

Neste módulo:
✅ O que é HTML e sua importância no desenvolvimento web.
✅ Estrutura básica de um arquivo HTML.
✅ Principais **tags** e **atributos** do HTML.
✅ Como adicionar **textos**, **cores** e outros elementos fundamentais.

---

### 🎨 **Fundamentos CSS**
📦 **Módulo**
- 🎥 **33 aulas**
- 📝 **4 Atividades**
- ⏳ **Duração: 2h 10min**

Neste módulo:
✅ O que é **CSS** e como ele funciona.
✅ Como adicionar estilos em uma página web.
✅ Conceito de **Box Model** e como ele influencia no layout.
✅ Estilização de **textos**, **cores** e **backgrounds**.

---

### 🍽️ **Projeto Prático: Página de Receita**
📦 **Módulo**
- 🎥 **10 aulas**
- ⏳ **Duração: 56min**

Chegou a hora de **colocar em prática** tudo o que foi aprendido!
✅ Desenvolvimento de uma **página de receita** do zero.
✅ Aplicação dos conceitos de **HTML e CSS**.
✅ Organização e estruturação de código.

---

## 🎯 **Objetivo do Curso**

Ao final deste curso, você será capaz de:
✔️ Criar páginas HTML estruturadas corretamente.
✔️ Estilizar páginas utilizando CSS com boas práticas.
✔️ Aplicar os conceitos fundamentais de **design responsivo**.
✔️ Desenvolver um **projeto prático** aplicando todo o conhecimento adquirido.
